# Apex Learning <img width="36" height="32" alt="logo_apex" src="https://github.com/user-attachments/assets/688f2937-fe32-422d-b275-737846106088" />

This project is a fully functional e-learning platform that provides a virtual classroom to connect students and teachers together. In 2023 was when i built this project as a final year project in Babcock University. This repo is the frontend of the project. Check out the backend -> [https://github.com/Xcoder-Creator/apex-learning-backend](https://github.com/Xcoder-Creator/apex-learning-backend).

## How to Run
1. Clone the repository to your local machine.
2. Open the project folder in VSCode.
3. Enter the terminal and run `npm run dev`.
4. Go to your browser and load `http://localhost:4500`.

## â—Important Note
Since the project was built in 2023, its running an old version of Node, specifically Node.js 16. Trying to run this project with anything newer than Node.js 16
will result in an openSSL error `Error: error:0308010C:digital envelope routines::unsupported` caused by Node.js 17+ switching to OpenSSL 3. So please install and use
Node.js 16 in other for the project to work on your machine. You can use `nvm` -> [https://www.nvmnode.com/guide/download.html](https://www.nvmnode.com/guide/download.html) to
switch to different Node.js versions.

## Tech Stack
- Vue JS
- Vuetify
- HTML
- CSS
- Javascript

## Features
- Responsive layout
- Clean and simple UI design
- Account creation and authentication for both students and teachers
- Virtual classroom creation by teachers
- Students join virtual classroom through classroom codes
- Classwork and attendance creation by teachers with deadline
- Students reponses to attendance and classwork is autograded by the system
